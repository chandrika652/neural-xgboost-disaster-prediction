# Neural XGBoost – Hybrid Disaster Prediction System

## Overview
This project proposes a hybrid machine learning framework combining neural networks and XGBoost to improve disaster prediction accuracy and risk assessment.

## Problem Statement
Accurate disaster prediction is essential for early warning systems and risk mitigation. Traditional statistical methods often fail to capture complex nonlinear patterns.

## Proposed Solution
A hybrid model where:
- Neural Network extracts nonlinear feature representations
- XGBoost performs final prediction using learned features
